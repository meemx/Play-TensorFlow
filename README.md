### VGGNet16
> + 该项目使用VGG16实现了图片的分类
> + 来源与参考：https://www.cs.toronto.edu/~frossard/post/vgg16/
> + vgg16v2.py将原作者代码vgg16.py更新为TensorFlow2.x下的代码
> + 训练完成的模型权重vgg16_weights.npz由于文件较大，请在此处下载：https://www.cs.toronto.edu/~frossard/post/vgg16/
